# 닐슨 코리아 시청기록 편집규칙 검증 연구

[![R](https://img.shields.io/badge/R-4.x-blue)](https://www.r-project.org)
[![Type](https://img.shields.io/badge/유형-산학협력-green)]()
[![Data](https://img.shields.io/badge/데이터-대외비-red)]()

닐슨 코리아로부터 샘플 원시 데이터와 편집 규칙 문서를 직접 제공받아, 편집 규칙 알고리즘을 R로 구현하고 공식 시청 데이터와 레코드 단위로 비교 검증한 산학협력 연구.

---

## 프로젝트 배경

시청률 데이터는 방송사의 콘텐츠 기획과 광고주의 매체 집행 의사결정에 직접적인 영향을 미치는 핵심 지표.

TV 시청 기록은 패널 가구에 설치된 미터기 장치를 통해 초 단위 로그로 수집. 이 원시 로그는 조사기관의 편집 규칙 알고리즘을 거쳐 분석 가능한 시청 데이터로 변환. 이 변환 과정은 외부에서 블랙박스로 존재. 편집 규칙이 정확히 적용되고 있는지, 변환 전후 데이터가 일관되게 유지되는지를 독립적으로 확인하기 어려움.

산학협력을 통해 닐슨 코리아(Nielsen Korea)로부터 원시 미터기 데이터와 편집 규칙 문서를 직접 제공받아, 연구실에서 독립적으로 알고리즘을 구현하고 공식 변환 결과와 비교 검증.

---

## 문제 상황

조사기관에서 제공하는 최종 시청 데이터는 외부에서 생성 과정을 검증하기 어려움. 편집 규칙 적용 기준이 내부 알고리즘에 의존하는 구조라, 변환 과정에서 기록 누락이나 채널 오매핑 같은 오류가 발생해도 외부에서는 발견하기 어려움.

시청 기록의 오차는 시청률과 시청 점유율 산출에 직접 영향을 미치며, 이는 방송사와 광고주의 의사결정 오류로 이어질 수 있음.

이 연구는 연구진이 편집 규칙 알고리즘을 직접 구현하고 공식 데이터와 비교해 불일치 패턴을 체계적으로 분류하는 방식으로 이 문제에 접근.

---

## 분석 및 구현 과정

텍스트 파일 형태의 원시 미터기 로그를 R로 파싱해 데이터프레임으로 변환, 비정상 시청 패턴을 탐지하는 편집 규칙 4종 구현. 구현 결과와 닐슨의 공식 TX3 데이터를 레코드 단위로 비교해 불일치 유형 분류 및 원인 분석.

```r
# 연구진 구현 결과와 TX3 데이터를 레코드 단위로 비교

# 완전 일치하는 기록 추출
merged_data <- merge(combined_df_filtered, tx3_df_filtered,
                     by = c("id", "channel_code", "start", "end", "date"),
                     all = FALSE)

# 연구진 결과에만 존재하는 기록
combined_only <- anti_join(combined_df_filtered, tx3_df_filtered,
                            by = c("id", "channel_code", "start", "end", "date"))

# 닐슨 측 결과에만 존재하는 기록
tx3_only <- anti_join(tx3_df_filtered, combined_df_filtered,
                      by = c("id", "channel_code", "start", "end", "date"))

# 시작 시각 불일치 기록
start_mismatch <- combined_df_filtered %>%
  inner_join(tx3_df_filtered, by = c("id", "channel_code", "end", "date")) %>%
  filter(start.x != start.y)

# 종료 시각 불일치 기록
end_mismatch <- combined_df_filtered %>%
  inner_join(tx3_df_filtered, by = c("id", "channel_code", "start", "date")) %>%
  filter(end.x != end.y)
```

| 분석 대상 | 하루 평균 약 3,500가구 규모의 샘플 데이터 (전체 패널 데이터가 아닌 샘플 제공) |
| 구현 편집 규칙 | 4종 비정상 시청 패턴 탐지 규칙 |
| 발견 불일치 유형 | 5종 |

### 5가지 불일치 패턴 요약

| 패턴 | 내용 |
|------|------|
| 닐슨 측 결과에만 존재 | 연구진 구현 결과에는 없지만 닐슨 공식 데이터에는 포함된 시청 기록 |
| 연구진 측 결과에만 존재 | 닐슨 공식 데이터에는 없지만 연구진 구현 결과에는 포함된 시청 기록 |
| 시작 시각 불일치 | 채널·종료 시각은 일치하지만 시청 시작 시각이 다르게 집계된 기록 |
| 종료 시각 불일치 | 채널·시작 시각은 일치하지만 시청 종료 시각이 다르게 집계된 기록 — 수분~수십 분 규모의 시청 시간 누락 케이스 확인 |
| 채널 코드 불일치 | 시청 시간은 일치하지만 채널 코드가 다르게 집계된 기록 — A 채널, B 채널 등 코드 이력 관리 문제로 발생 |

## 향후 연구 방향

- VOD 시청 로그와 실시간 채널 로그 구분을 위한 추가 정보 확보 및 처리 로직 개선
- 채널 코드 이력을 시점별로 관리하는 매핑 체계 구축 방안 탐색
- 미터기 데이터와 공식 변환 결과 간 자동화 비교 검증 파이프라인 개발

---

## 역할

- 3인 팀에서 연구 전체 주도 — 데이터 전처리, 알고리즘 구현, 비교 검증, 보고서 작성
- 비정형 초 단위 미터기 로그 데이터를 R로 파싱해 분석 가능한 데이터프레임으로 정형화
- 팀 내 R 코드 구현 및 검증 총괄

---

## 적용 가능성

- 방송 시청 조사 기관의 편집 규칙 검증 체계 정례화
- 외부 감사·제3자 검증 파이프라인으로 시청 데이터 신뢰성 확보
- 원시 미터기 로그 → 집계 데이터 간 자동 일관성 검사 도구로 확장 가능
- 동일 검증 프레임워크를 VOD·OTT 시청 데이터로 적용 가능

---

## 데이터 안내

닐슨 코리아 원시 가구 시청 데이터는 대외비로 이 레포에 포함되어 있지 않음. 검증 방법론만 공개.
